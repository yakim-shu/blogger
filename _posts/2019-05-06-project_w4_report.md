## 2019/05/12
#### 本日進度
- 寫完 Week5 心得

#### 心得
Week5 碎念心得再寫下去可能會吐血，交完後決定今天就這樣，不再碰課程。

下午放風時間，又繞到社頂公園騎一圈，機車熄火從山頂溜下去享受涼風，有山有海的墾丁其實真的很美。

因為惱人姨媽 + 天天下雨，已經一個禮拜沒跑步了，今天回歸運動的感覺爆爽，但爽感總是要熬過痛苦期才能擁有，得來十分費功夫。

「 恭喜你！你突破了最快速度 」偶爾結束紀錄 APP，都會給我類似的獎章，我會因為速度變快、距離變長而被稱讚，但我追求的從來不是更快。

每次都是為放慢腳步做練習、我只求讓痛苦減低、呼吸順暢，更重要的是不要放棄，如果是我開發  APP，可能會這設計個龜速獎章給自己：

「 矮油，今天速度很慢喔，但就算連操場健走的阿罵都追不上，你還是選擇來操場報到，你這傢伙簡直莫名其妙（ 啊不是！ ）你真的棒呆了 」

莫名惆悵： 明明每天星空都很美，但只有在跑步的晚上才會認真欣賞

#### 心得
昨晚才講著幹話，說一天洗 10 次澡會不會有靈感，結果晚上洗著洗著、靈感就找上門了，surprise!!!

## 2019/05/08
#### 本日進度
- 寫 API 筆記
- 裝好沒啥屁用的 live reload

#### 心得
寫完拖很久的 API 筆記，查了不少資料，邊寫邊統整，本來以為做筆記這件事，或許可以熟能生巧、越來越快，但發現這一個月以來，速度可是一點都沒有進步呢 (´･_･`)

今天在思考「 如何從旁敲側擊地增加寫筆記效率 」，熊熊想起一直忘了用 Live reload！每次寫完一段、想要預覽，就要不停按 `ctrl + R`，遲遲不更新的頁面就像打火機、一直試圖點燃我。

所以為了降低心中怒火，今天終於把自動刷新弄好，後來才發現 `jekyll` 比較新的版本其實有內建，天啊超級方便。

> 結果內建的 live reload 非常雞肋，修改完要等 4 秒才更新... 乾 超慢

#### 今天學到的
- `curl` 指令參數好多、滿複雜的，但功能很強大
- 想弄懂 JSON 跟 JS 物件的對照表，很推薦這網站：[JSON Editor online](https://jsoneditoronline.org)
- 「 非 」 RESTful API
    - **動詞是寫在 URL**
- RESTful API
    - **語易化的動詞 Method** + **名詞 URL**
