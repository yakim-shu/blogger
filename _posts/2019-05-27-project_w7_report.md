#### 交換資料 - AJAX 
- 不難、就是原生寫法有點繁瑣
- 一定會受到瀏覽器的同源政策影響，可使用 CORS 突破限制
    - Server 端 response 加上 `access-control-allow-origin: *`
- 請求又分 simple requests & Preflight Request
    - 是一個防護機制，`非 GET || 帶 Header 參數`的 request，都會發送一個「Preflight Request」，就會知道這個 API 有沒有提供 CORS。
    - 如果沒有的話，真正的 request 就會被擋下來，以防資料庫被亂搞。 
           
#### 交換資料 - JSONP
- 利用 `<script src="...">` 不受同源限制的特性，直接載入一隻帶參數的 js，當作是發 request，用一個 function 包裝起來。
- 缺點： 參數永遠都只能用附加在網址上的方式（GET）帶過去，沒辦法用 POST。

#### 第二期直播
- `let` & `const` 的 Temporal Dead Zone 特性真的蠻酷的，不曉得一個程式語言是以怎麼樣的基礎跟考量進行設計？
- 對 callback function 的理解又更深了一點
